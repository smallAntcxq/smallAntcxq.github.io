css之block,inline-block,inlien的区别
发布于：2016-12-05 09:32:32 标签：/ css / 访问：

1.inline

inline：使元素变成行内元素（内联），拥有行内元素的特性，即

  1. 与其他行内元素共享一行
  2. 不能修改width、height属性，大小由内容撑开
  3. padding属性 top、right、bottom、left设置都有效；margin属性只有left、right设置有效
  4. 可以设置verticle-align

2.block

block： 使元素变成块级元素，拥有块级的特性，即

  1. 独占一行，在不设置自己的宽度的情况下，块级元素会默认填满父级元素的宽度
  2. 可以修改width、height属性
  3. padding、margin四个方向的值设置均有效
  4. 不可以设置verticle-align

3.inline-block

inline-block： 使元素变成行内块级元素，结合了行内元素和块级元素的特性（不独占一行的块级元素），即

  1. 与其他行内元素共享一行
  2. 可以修改width、height属性
  3. padding、margin四个方向的值设置均有效
  4. 可以设置verticle-align

4.行内元素、块级元素区别

行内元素：在一行显示，不可以设置宽高。代表 span a select strong br

块级元素：换行显示，可以设置宽高。代表 div ul ol h1-h6 table

行内块级元素：在一行显示，可以设置宽高。代表 img input
