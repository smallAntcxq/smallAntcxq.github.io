    1. HTTP协议

    HTTP协议： 对客户端和服务器端之间数据传输的格式规范。

    HTTP协议约定的内容：

    HTTP协议是基于传输层的TCP协议，在每一次客户端和服务器通信之前都会去建立tcp连接(三次握手)

    HTTP协议的特点：

    1. 无状态
    2. 简单快速
    3. 灵活

    1.1 请求报文

    请求方式 ： get(获取资源) post(提交资源) put(更新资源) delete(删除资源)

    1.3 HTTPS

    HTTP协议特点：明文传输，在客户端和服务器通信过程中，数据传输是明文。

    HTTPS = HTTP + SSL/TLS(加密)

    我们可以理解为HTTPS是安全版的HTTP

    HTTP端口是80

    HTTPS端口是443
