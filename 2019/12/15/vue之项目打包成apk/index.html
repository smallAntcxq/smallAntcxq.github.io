vue之项目打包成apk
发布于：2019-12-15 17:30:19 标签：/ 项目打包 / 访问：
1.第一步（重要）

登录阿里云在远程服务器添加一个4000端口 ***

修改服务器上：C:\Windows\System32\drivers\etc中的hosts文件

注意：前者：服务器内网 后者：接口运行为：localhost:4000

在服务器端运行接口：node app(localhost:4000)与第三步对应

2.第二步
修改vue.config.js中的publicPath(解决资源找不到路径问题)
3.第三步

打包成app不存在跨域访问的问题

4.第四步

执行 npm run build 打包 成dist

5.第五步

打开hbuilder

​ 1.文件+新建+移动App（项目名）

​ 2.把项目中除了.project 和manifest.json的所有文件删除

​ 3.把dist中的文件拷贝到项目中

6.第六步

配置manifest.json文件

​ 修改应用信息

​ 修改图标配置

​ 修改启动图片

7.第七步

​ 打包发行

​ 项目右键+发行

8.第八步

​ 拷贝apk ,安装
